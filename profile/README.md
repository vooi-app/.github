## What VOOI is

VOOI is a unified perpetual execution layer for onchain trading. We aggregate Hyperliquid, Lighter, Aster, trade.xyz, and Kinetiq into a single non-custodial interface for three kinds of users:

- **Active traders** use VOOI Ultra to access every market on every venue from one app.
- **Builders** plug VOOI Perps API into their product instead of integrating five DEXs separately.
- **Strategists** automate funding arbitrage and cross-venue strategies through the same layer.

The execution layer is the same for all three. Trader on Ultra, builder on the API, agent on MCP — same liquidity, same margin, same rails.

## What you can do with VOOI

- **Trade 500+ markets from one app.** Crypto perps (BTC, ETH, SOL, HYPE, and 400+ more) and RWA perps (gold, oil, US stocks like NVDA, indices like S&P 500, Korean equities like SAMSUNG and HYUNDAI). Non-custodial throughout.
- **Build perps into your product with one integration.** VOOI Perps API exposes 40+ endpoints, one auth model, and ~100ms P50 response time. Route orders, transfer margin between venues, stream market data — without writing five integrations.
- **Move margin across venues in seconds.** Margin Transfer is an intent-based, non-custodial primitive that moves collateral between venues end-to-end in ~15 seconds. No bridging, no withdraw-redeposit cycle.
- **Automate funding arbitrage.** Compare funding rates across all five DEXs in real time. Long one venue, short another, capture the spread. Strategies can run via the API or autonomously via MCP.
- **Plug AI agents directly into trading.** VOOI Perps API is exposed as an MCP server. Connect Claude, GPT, or any MCP-compatible model and prompt multi-venue execution. Agents read markets, size positions, and route orders through the same infrastructure human traders use.
